CSS란?

  • CSS는 Cascading Style Sheets 의 약자입니다.
  • CSS는 한번에 여러 웹페이지(html파일)에 적용될 수 있습니다.
  • 확장자 css는 별도로 외부파일에 css를 작성할때 사용합니다.
  • CSS 작성은 style 태그 사용합니다. 특정 요소에 적용할 선택자(selector)가 필요합니다.
  • CSS 프로퍼터는 글꼴 크기,색상, 배경이미지, 테두리 등등이 있습니다. 모든 태그에 공통적인것 또는 특정태그에만 적용되는 것을 구별해서 사용합니다.
